
# BilibiliDown 是什么

BilibiliDown 是一款专注于下载[Bilibili](https://www.bilibili.com)上精彩视频的下载工具。  
它具有以下丰富的特性：  
-  支持跨平台(Windows/Linux/MacOS)  
-  支持GUI图形人机交互界面  
-  支持扫码/密码/短信登录  
-  支持各种链接解析
-  支持自定义各种链接解析
-  支持收藏夹下载  
-  支持稍后再看下载  
-  支持UP主所有视频/合集/视频列表下载  
-  支持断点续传下载(因异常原因退出后, 只要下载目录不变, 直接在上次基础上继续下载)
-  支持CC字幕下载  
-  支持弹幕下载  
-  支持互动视频下载  
-  支持课程视频下载  
-  支持音频下载  
-  支持图文动态的图片下载
-  支持专栏的图片下载
-  支持专栏文集解析
-  支持图文动态个人收藏解析
